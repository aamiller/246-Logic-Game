Instructions: This is a logic game where there is a hidden rule that fits some triplets of numbers and not others. You guess numbers by inputting a number and testing if it fits the rule. The base case that is given fits the rule. When you've performed all the further experimental tests you want, you can test what you think the rule is against 5 cases. If you correctly identify each case as fitting with the rule (or not), you pass the level.
